Switch

switch (day) {
  case 1: cout << "Mon"; break;
  case 2: cout << "Tue"; break;
  default: cout << "Other";
}
⚠️ Forget break = falls through to next case! Always add it.

Arrays

1D Array 
int arr[5] = {10,20,30,40,50};
//index:  [0][1][2][3][4]

arr[0] → 10 (first)
arr[4] → 50 (last)
2D Array 
int mat[3][3] = {
  {1, 2, 3},
  {4, 5, 6},
  {7, 8, 9}
};
mat[1][2] → 6
Zero-Indexed Contiguous Memory Passed by Reference
☝️ first element is always arr[0], not arr[1]!

Pass by Value vs Reference

❌ Pass by Value

void f(int x)

Gets a COPY

Original stays UNCHANGED

void inc(int x) { x++; }
int a = 5;
inc(a);
// a is still 5 😢

✅ Pass by Reference

void f(int &x)

Gets the ORIGINAL

Original gets MODIFIED

void inc(int &x) { x++; }
int a = 5;
inc(a);
// a is now 6 🎉
⚡ Arrays are ALWAYS passed by reference — no & needed. Any change inside function changes the original array!
