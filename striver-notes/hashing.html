What is Hashing?

Pre-store data → fetch instantly in O(1). Instead of searching every time, store frequencies once.

// Brute: each query loops array → O(Q × N) ❌
// Hashing: pre-compute once → O(N + Q) ✅
Count once, answer any query instantly 🪄

Number Hashing

int hash[13] = {0};
for(int i=0; i<n; i++)
    hash[arr[i]]++;

hash[1]  // → freq of 1
hash[4]  // → 0 if absent
index = number, value = count
🔤

Character Hashing

Lowercase (26) 
int hash[26]={0};
hash[s[i] - 'a']++;
// 'a'→0, 'b'→1...
All ASCII (256) 
int hash[256]={0};
hash[s[i]]++;
// auto-cast to ASCII
💡No size issues — max 256 chars exist. Always use arrays for char hashing.
🗺️

⚡Strategy: Default to unordered_map (O(1)). If TLE from collisions → switch to map.
💥

Collisions & Division Method

Internal hashing: hash_index = key % table_size

// table_size = 10
28 % 10 = 8
38 % 10 = 8  ← COLLISION!
48 % 10 = 8  ← COLLISION!

// Linear chaining:
Index 8: [28] → [38] → [48]
⚠️Worst case O(N) when all keys hash to same index. Extremely rare in practice.

FAANG Interview Questions

Two Sum Everyone 
vector<int> twoSum(vector<int>& a, int t) {
    unordered_map<int,int> mp;
    for(int i=0; i<a.size(); i++) {
        int c = t - a[i];
        if(mp.count(c)) return {mp[c], i};
        mp[a[i]] = i;
    }
    return {};
}
Valid Anagram Meta MS 
bool isAnagram(string s, string t) {
    if(s.size()!=t.size()) return false;
    int f[26]={0};
    for(int i=0; i<s.size(); i++)
        { f[s[i]-'a']++; f[t[i]-'a']--; }
    for(int x : f) if(x) return false;
    return true;
}
Subarray Sum = K Google Amazon 
int subarraySum(vector<int>& a, int k) {
    unordered_map<int,int> m; m[0]=1;
    int sum=0, cnt=0;
    for(int x : a) {
        sum += x;
        if(m.count(sum-k)) cnt += m[sum-k];
        m[sum]++;
    }
    return cnt;
}
Theory All
  • map: sorted, O(log N), any key
  • unordered_map: O(1) avg, basic keys
  • Default unordered, switch if TLE
  • Collision = same hash index → O(N)

⭐ Remember

Pre-store + fetch O(1). Array hash for small numbers, map for large. unordered_map by default — switch to map only if collisions cause TLE.
