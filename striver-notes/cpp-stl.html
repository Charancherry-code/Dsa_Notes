Pairs

pair<int, int> p = {1, 3};

p.first   // → 1
p.second  // → 3

// Nested: store 3 values
pair<int, pair<int,int>> p = {1, {3,4}};
p.second.first  // → 3
p.second.second // → 4
Pair = store 2 things together 🤝

Vector — Dynamic Array

Declare & Insert 
vector<int> v;
v.push_back(1);
v.emplace_back(2); // faster!

vector<int> v(5, 100);
// {100,100,100,100,100}

vector<int> v2(v); // copy
Access & Delete 
v[0]       // first element
v.back()   // last element
v.size()   // count
v.pop_back() // remove last
v.clear()  // remove all
v.empty()  // is empty?

v.erase(v.begin()+1);
// deletes index 1
⚡ emplace_back > push_back — constructs in-place, no copy overhead
👆

Iterators

Points to memory address of element. Use *it to get value.

auto it = v.begin();  // points to first
*it                    // → value at that address
it++                   // move to next element

v.begin()  // → first element
v.end()    // → AFTER last (not last!)
begin → first end → after last *it → get value it++ → next
auto = compiler figures out the data type for you 🪄
🖨️

Print Vector

// ① Index loop
for(int i=0; i<v.size(); i++)
    cout << v[i];

// ② Iterator
for(auto it=v.begin(); it!=v.end(); it++)
    cout << *it;

// ③ For-each (cleanest ✨)
for(auto x : v)
    cout << x;

Stack — LIFO

stack<int> st;
st.push(1);  st.push(2);  st.push(3);

st.top()   // → 3 (peek)
st.pop()   // removes 3
st.top()   // → 2 now
st.size()  // → 2
Last In First Out All O(1)
❌No indexing! Only top/push/pop
🚶

Queue — FIFO

queue<int> q;
q.push(1);  q.push(2);  q.push(4);

q.front()  // → 1
q.back()   // → 4
q.pop()    // removes front (1)
q.front()  // → 2 now
First In First Out All O(1)
⛰️

Priority Queue

// Max Heap (default)
priority_queue<int> pq;
pq.push(5); pq.push(2); pq.push(10);
pq.top()  // → 10 (largest)

// Min Heap
priority_queue<int,vector<int>,
    greater<int>> pq;
pq.top()  // → smallest
OpTime
pushO(log N)
topO(1)
popO(log N)

Set — Sorted + Unique

set<int> s;
s.insert(1);  // {1}
s.insert(4);  // {1,4}
s.insert(2);  // {1,2,4} auto sorted!
s.insert(2);  // {1,2,4} dup ignored!

s.find(3)    // → s.end() (not found)
s.find(2)    // → iterator to 2
s.count(2)   // → 1 (exists)
s.erase(2)   // removes 2
VariantSortedUniqueTime
set✅✅O(log N)
multiset✅❌O(log N)
unordered_set❌✅O(1) avg
💡 multiset: erase(element) removes ALL. Use erase(find(x)) to remove ONE.

Map — Key:Value Pairs

Keys are sorted & unique. Think: roll number → name.

map<int, int> mp;
mp[1] = 2;            // key:1 val:2
mp.emplace(3, 1);     // key:3 val:1
mp.insert({2, 4});    // key:2 val:4
// stored: {1:2, 2:4, 3:1}

mp[1]      // → 2
mp[5]      // → 0 (doesn't exist)
mp.find(3) // → iterator to {3,1}
// Iterate
for(auto it : mp) {
    cout << it.first << " "
         << it.second;
}
// prints in sorted key order!
VariantKeysTime
mapsorted uniqueO(log N)
multimapsorted dupsO(log N)
unordered_mapunique O(1)O(1) avg

Sort

sort(a, a+n);           // ascending
sort(v.begin(), v.end());

// Descending
sort(a, a+n, greater<int>());

// Partial sort
sort(a+2, a+4);
// only sorts index [2,4)
O(N log N) — one line replaces bubble/selection sort!
⚙️

Custom Comparator

Sort YOUR way — "Is p1 before p2 correct? return true/false"

bool comp(pair<int,int> p1, pair<int,int> p2) {
    if(p1.second < p2.second) return true;   // sort by second ↑
    if(p1.second > p2.second) return false;
    // if second same → sort by first descending
    return p1.first > p2.first;
}

sort(arr, arr+n, comp);  // pass as 3rd argument
💡 Trick: Think of only 2 elements. "Should p1 come before p2?" If yes → true. If no → false. That's it.
