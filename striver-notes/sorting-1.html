Selection Sort

Select the minimum, swap to front. Repeat.

void selectionSort(int arr[], int n) {
    for(int i=0; i<n-1; i++) {
        int mini = i;
        for(int j=i+1; j<n; j++)
            if(arr[j] < arr[mini]) mini = j;
        swap(arr[mini], arr[i]);
    }
}
Always O(N²)Unstable
2️⃣

Bubble Sort

Adjacent swaps, max bubbles to end.

for(int i=n-1; i>=1; i--) {
  bool swapped=false;
  for(int j=0; j<i; j++)
    if(arr[j]>arr[j+1]) {
      swap(arr[j],arr[j+1]);
      swapped=true;
    }
  if(!swapped) break;
}
Best O(N)Stable
3️⃣

Insertion Sort

Place each element in its correct spot.

for(int i=1; i<n; i++) {
  int j=i;
  while(j>0 &&
        arr[j-1]>arr[j]) {
    swap(arr[j-1],arr[j]);
    j--;
  }
}
Best O(N)Stable

Common Mistakes

❌Bubble without swapped flag (no O(N))
❌Insertion j>=0 (accesses arr[-1]!)
❌Selection: swap inside inner loop
❌Saying selection is stable (it's NOT)

K-positions-away sort Google

Use Insertion → inner loop runs ≤ K times → O(N×K).

⭐ Remember

Selection: always N², unstable. Bubble & Insertion: O(N) best, stable. For large data, never use these — go N log N (Merge/Quick).
